# ⚡ QuizNova

A premium, modern, and fully responsive Quiz Web App built with **HTML**, **Tailwind CSS**, and **JavaScript**.  
QuizNova delivers a fast-paced quiz experience with animated UI, timer-based gameplay, score tracking, multiple categories, difficulty levels, and smooth transitions.

## 🌐 Live Demo
🔗 https://shankar2006458.github.io/QuizNova/

---

## ✨ Features

### 🎮 Core Gameplay
- Multiple quiz categories:
  - General Knowledge
  - Coding
  - Movies
  - Sports
  - Science
- Difficulty modes:
  - Easy
  - Medium
  - Hard
- Question count selector:
  - 5 / 10 / 15 / 20
- 15 second timer for every question
- Skip question option
- Instant answer validation
- Auto next question flow

### 🏆 Score System
- +10 points for every correct answer
- +5 speed bonus for fast answers
- Accuracy percentage
- Correct / Wrong / Skipped stats
- Best streak tracker
- Average answer time
- Best score saved in LocalStorage

### 🎨 Premium UI / UX
- Glassmorphism interface
- Animated gradient background
- Smooth transitions
- Ripple click effects
- Modern cards and buttons
- Responsive for mobile + desktop
- Popup modals for rules, pause, results
- Lucide Icons integration

### ⚡ Smart Features
- Tab switch detection (auto pause)
- Replay quiz instantly
- Quick Play mode
- Saved settings in browser
- Unique shuffled questions every game
